# 🌰 DAWSONS - Dry Fruits Website

A modern and responsive e-commerce website for **DAWSONS Dry Fruits**, developed using **HTML, CSS, and JavaScript**. The website provides an attractive user interface for customers to explore products, manage their shopping cart, and place orders.

---

## 📌 Features

- 🏠 Home Page
- 🛍 Product Listing
- 🛒 Shopping Cart
- 💳 Checkout Page
- 🔐 Login & Signup Pages
- 📞 Contact Us Page
- 🔍 Product Search
- 📱 Responsive Design
- 🎨 Modern UI

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub

## 🚀 How to Run the Project

### 1. Clone the repository

```bash
git clone https://github.com/Nadiajabbar/DAWSONS-Website.git
```

### 2. Open the project folder

```bash
cd DAWSONS-Website
```

### 3. Run the website

Simply open **index.html** in your web browser.
